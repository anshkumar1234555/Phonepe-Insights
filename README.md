PhonePe Transaction Insights

Project Overview
PhonePe Transaction Insights is a data analytics project developed to analyze digital payment transaction trends using the PhonePe Pulse dataset. The project focuses on extracting, transforming, and analyzing transaction data to generate meaningful business insights and visualize them through an interactive Streamlit dashboard.

Features
Data extraction from JSON datasets
Data cleaning and preprocessing
SQL database integration using SQLite
Exploratory Data Analysis (EDA)
Interactive Streamlit dashboard
Transaction trend analysis
Business insights generation
